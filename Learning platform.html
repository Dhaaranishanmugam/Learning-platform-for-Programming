Fibonacci Series

  • Initialize Variables: Set the first two numbers of the series as 0 and 1.
  • Iterate: Loop through the desired range.
  • Update Series: Add the last two numbers to get the next number in the series.
  • Store & Print: Store the new number and print the series.


Factorial of a Number

  • Initialize Result: Set the result as 1.
  • Loop: Iterate from 1 to the given number.
  • Multiply: Multiply the result by the current iteration number.
  • Print Result: After the loop, print the result.


Strong Number

  • Initialize Sum: Set the sum of factorial of digits to 0.
  • Extract Digits: Loop through each digit of the number.
  • Calculate Factorial: Compute the factorial of each digit and add to the sum.
  • Compare: Compare the sum to the original number to determine if it is a strong number.


Palindrome of a Number

  • Initialize Reverse: Set the reverse of the number to 0.
  • Extract Digits: Loop through each digit of the number.
  • Form Reverse: Multiply the reverse by 10 and add the extracted digit.
  • Compare: Compare the reverse to the original number to check for a palindrome.


Factors of a Number

  • Initialize List: Create an empty list to store factors.
  • Loop: Iterate from 1 to the given number.
  • Check Divisibility: For each iteration, check if the number is divisible.
  • Store & Print: If divisible, add to the list and print all factors at the end.


Prime Number

  • Check for Edge Cases: If the number is less than 2, it is not prime.
  • Initialize Loop: Loop from 2 to the square root of the number.
  • Check Divisibility: If the number is divisible by any number in the loop, it is not prime.
  • Determine Primality: If no divisors are found, the number is prime.


Sum of Number till Single Digit

  • Initialize Sum: Set the initial sum to 0.
  • Extract Digits: Loop through each digit of the number, adding them to the sum.
  • Reduce Sum: If the sum is greater than 9, repeat the process on the sum.
  • Return Single Digit: When the sum is a single digit, print it.


Reverse the Number

  • Initialize Reverse: Set the reverse of the number to 0.
  • Extract Digits: Loop through each digit of the number.
  • Form Reverse: Multiply the reverse by 10 and add the extracted digit.
  • Print Reverse: After processing all digits, print the reversed number.


Hollow Square pattern

  • Initialize Dimensions: Set the dimensions for the square (n x n).
  • Loop Through Rows: Loop from 1 to n for the rows.
  • Loop Through Columns: For each row, loop from 1 to n for the columns.
  • Print Pattern: Print '*' for the borders (first and last row/column), and print ' ' (space) for the hollow part.


Palindrome of a string

  • Convert Case: Convert the string to the same case (lowercase).
  • Initialize Pointers: Set two pointers, one at the start and one at the end of the string.
  • Compare Characters: Loop through the string, comparing characters at the two pointers.
  • Check Palindrome: If all characters match, the string is a palindrome; otherwise, it is not.
